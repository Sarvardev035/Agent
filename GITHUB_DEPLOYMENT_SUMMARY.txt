🚀 DEPLOYING TO VERCEL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Step 1: Connect to Vercel (5 minutes)
  1. Go to https://vercel.com
  2. Click "New Project"
  3. Select "Import Git Repository"
  4. Paste: https://github.com/Sarvardev035/Agent.git
  5. Click "Import"

Step 2: Configure Build Settings
  Framework:        Vite
  Root Directory:   ./
  Build Command:    npm run build
  Output Directory: dist
  Install Command:  npm install

Step 3: Add Environment Variables
  VITE_EXCHANGE_API_KEY = your_api_key_from_exchangerate_api_com
  VITE_API_URL = https://finly.uyqidir.uz (optional)

Step 4: Deploy
  1. Click "Deploy"
  2. Wait for build (~2-3 minutes)
  3. You'll get a Vercel URL

Step 5: Configure Custom Domain (inteullpo.online)
  1. Project Settings → Domains
  2. Add Domain: inteullpo.online
  3. Copy CNAME record
  4. Add to domain registrar's DNS
  5. Wait 24-48 hours for propagation
